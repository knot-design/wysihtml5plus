Features

  • Easy to use.
  • Fast and lightweight.
  • Auto-linking of urls as-you-type.
  • Generates valid and semantic HTML5 markup.
  • Various languages.
  • Image Uploader/Manager included.
  • Speech-input for Chrome.

  1. Include CSS and scripts

    <!-- CSS -->
<link href="/path-to-wysihtml5plus/dist/wysihtml5plus.min.css" rel="stylesheet"  type="text/css">
<!-- Library -->
<script src="/path-to-wysihtml5plus/dist/wysihtml5plus.min.js"></script>
<!-- wysihtml5 parser rules -->
<script src="/path-to-wysihtml5plus/parser_rules/advanced.js"></script>

  2. Create a textarea

    <textarea id="wysiwyg-area" placeholder="Enter your text ..."></textarea>

  3. Initialize

    var editor = new wysihtml5.Editor("wysiwyg-area", { // id of element
  toolbar:  {
    button: ['bold', 'italic', 'block', 'fontSize', 'colors', 'ul', 'ol', 'align', 'link', 'unlink', 'undo', 'redo']
  },
  parserRules:  wysihtml5ParserRules // defined in parser rules set
});
